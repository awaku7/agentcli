# uag (uagent)

uag — это универсальный агент выполнения инструментов, работающий в вашей локальной среде. Он взаимодействует с пользователями через интерфейс командной строки (CLI) и выполняет различные задачи, такие как операции с файлами, поиск в Интернете и выполнение сценариев Python в соответствии с инструкциями.

## Основные функции

- **Локальные операции с файлами**: чтение, запись, редактирование и поиск файлов.
- **Поиск информации**: веб-поиск с помощью DuckDuckGo и извлечение содержимого веб-страниц.
- **Выполнение кода**: безопасный запуск сценариев Python и команд PowerShell.
- **Мультимедийная обработка**: генерация изображений, чтение файлов PDF/PPTX, создание скриншотов.
- **Многоязычная поддержка**: поддержка нескольких языков, включая русский, японский и английский.
- **Поддержка MCP (Model Context Protocol)**: возможность подключения к внешним серверам MCP для расширения функций.

## Установка

Вы можете установить его с помощью pip из PyPI:

```bash
pip install uag
```

При первом запуске автоматически запустится мастер настройки.

## Быстрый старт

После установки просто введите следующую команду для запуска:

```bash
uag
```

После запуска вы можете просить агента о таких вещах, как:
- "Прочитай README в текущем каталоге и кратко изложи его содержание."
- "Найди в Интернете последние новости об ИИ и сделай краткий обзор."
- "Сжми все PNG-файлы в папке 'images' в ZIP-архив."
