# uag (uagent)

uag เป็นเอเจนต์เรียกใช้เครื่องมืออเนกประสงค์ที่ทำงานในสภาพแวดล้อมท้องถิ่นของคุณ โดยจะโต้ตอบกับผู้ใช้ผ่านอินเทอร์เฟซบรรทัดคำสั่ง (CLI) และทำงานต่างๆ เช่น การจัดการไฟล์ การค้นหาเว็บ และการรันสคริปต์ Python ตามคำสั่ง

## คุณสมบัติหลัก

- **การจัดการไฟล์ในเครื่อง**: อ่าน เขียน แก้ไข และค้นหาไฟล์
- **การดึงข้อมูล**: ค้นหาเว็บด้วย DuckDuckGo และดึงเนื้อหาจากหน้าเว็บ
- **การรันโค้ด**: รันสคริปต์ Python และคำสั่ง PowerShell อย่างปลอดภัย
- **การประมวลผลมัลติมีเดีย**: สร้างรูปภาพ อ่านไฟล์ PDF/PPTX จับภาพหน้าจอ
- **การรองรับหลายภาษา**: รองรับหลายภาษา รวมถึงภาษาไทย ภาษาญี่ปุ่น และภาษาอังกฤษ
- **การรองรับ MCP (Model Context Protocol)**: สามารถเชื่อมต่อกับเซิร์กเวอร์ MCP ภายนอกเพื่อขยายความสามารถ

## การติดตั้ง

คุณสามารถติดตั้งโดยใช้ pip จาก PyPI:

```bash
pip install uag
```

เมื่อรันครั้งแรก ตัวช่วยตั้งค่าจะเริ่มทำงานโดยอัตโนมัติ

## เริ่มต้นอย่างรวดเร็ว

หลังจากการติดตั้ง เพียงพิมพ์คำสั่งต่อไปนี้เพื่อเริ่มต้น:

```bash
uag
```

เมื่อเริ่มทำงานแล้ว คุณสามารถขอให้เอเจนต์ทำสิ่งต่างๆ เช่น:
- "อ่าน README ในไดเรกทอรีปัจจุบันและสรุปเนื้อหา"
- "ค้นหาข่าว AI ล่าสุดบนเว็บและทำสรุป"
- "บีบอัดไฟล์ PNG ทั้งหมดในโฟลเดอร์ images เป็นไฟล์ ZIP"
