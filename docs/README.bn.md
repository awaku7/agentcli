## কেন?

**বিক্রেতা লক-ইন থেকে বিরত থাকুন।** বেশিরভাগ AI সহকারী আপনাকে একটি নির্দিষ্ট প্রদানকারী বা ক্লাউড পরিষেবার সাথে সংযুক্ত করে। uag ভিন্ন।

- **স্থানীয়ভাবে চলে** আপনার মেশিনে। আপনার ডেটা আপনার সাথে থাকে (আপনি যে API কলগুলি করেন তা ছাড়া)।
- **প্রোভাইডার স্বাধীনতা**: OpenAI, Claude, Gemini, DeepSeek, Ollama, Azure, Bedrock, HuggingFace... 15+ প্রদানকারী, সমস্ত একটি একক ইন্টারফেস থেকে অ্যাক্সেসযোগ্য। এনভায়রনমেন্ট ভেরিয়েবল পুনরায় কনফিগার করে তাদের মধ্যে অদলবদল করুন — কোন রিইন্সটল নেই, মাইগ্রেশন নেই।
- **131 টুল**: ফাইল I/O, ওয়েব সার্চ, ইমেজ জেনারেশন, Gmail, BLE ডিভাইস স্ক্যানিং, MCP সার্ভার ইন্টিগ্রেশন — **76 সমান্তরাল-নিরাপদ** (8 পর্যন্ত থ্রেড পুলের মাধ্যমে একযোগে কার্যকর করা যায়, `UAGENT_PARALLEL_WORKERS` এর মাধ্যমে কনফিগার করা যায়)। যখন LLM একাধিক টুল কল একবারে ফায়ার করে, uag স্বয়ংক্রিয়ভাবে তাদের সমান্তরাল করে।
- **3 UI + A2A**: CLI, GUI, ওয়েব এবং এজেন্ট-টু-এজেন্ট প্রোটোকল। একই ইঞ্জিন, যেকোনো ইন্টারফেস।
- **IoT প্রস্তুত**: SwitchBot, ECHONET Lite, Matter, UPnP — AI এর মাধ্যমে আপনার বাড়ির ডিভাইসগুলি নিয়ন্ত্রণ করুন৷
- **এজেন্ট দক্ষতা**: মার্কেটপ্লেস থেকে কমিউনিটি-নির্মিত দক্ষতা ইনস্টল করুন। অবিরামভাবে uag প্রসারিত করুন.

uag হল **আপনার শর্তে আপনার এআই সহকারী**। একটি প্রদানকারীর সাথে আবদ্ধ নয়, একটি ইন্টারফেসের সাথে আবদ্ধ নয়, একটি প্ল্যাটফর্মের সাথে আবদ্ধ নয়।

## বৈশিষ্ট্য

### 🧠 মাল্টি-প্রোভাইডার আর্কিটেকচার

OpenAI / Azure / Bedrock / OpenRouter / Ollama / Gemini / Vertex AI / Claude / Grok / NVIDIA / DeepSeek / Z.AI (Zhipu AI) / HuggingFace / Alibaba ক্লাউড (Qwen) / KIMI (Moonshot AI) / Xiaomi MiMo / LMakguana / MinFa ** স্টুডিও **

সমস্ত প্রদানকারী একই টুলসেট এবং ইন্টারফেস ভাগ করে। `UAGENT_PROVIDER` সেট করে স্যুইচ করুন — কোনো কোড পরিবর্তন নেই, কোনো পৃথক ইনস্টলেশন নেই৷

### ⚡ সমান্তরাল টুল এক্সিকিউশন

যখন LLM একই সাথে একাধিক টুলের অনুরোধ করে, uag **স্বয়ংক্রিয়ভাবে তাদের সমান্তরাল করে তোলে।
76টি টুলগুলিকে `x_parallel_safe` চিহ্নিত করা হয়েছে এবং একটি `ThreadPoolExecutor` এর মাধ্যমে একযোগে চালান (ডিফল্টভাবে ৮টি থ্রেড; পরিবর্তন করতে `UAGENT_PARALLEL_WORKERS` সেট করুন)।

**উদাহরণ**: "নর্ডিক রাজধানীতে আবহাওয়া পরীক্ষা করুন" জিজ্ঞাসা করুন → LLM ফায়ার `সার্চ_ওয়েব` × 5টি দেশ → সমস্ত 5টি অনুসন্ধান সমান্তরালভাবে চলে → একটি ব্যাচে সংগৃহীত ফলাফল।

শুধুমাত্র-পঠন সরঞ্জাম (ফাইল অনুসন্ধান, হ্যাশ গণনা, ডিরেক্টরি তালিকা, অনুবাদ, DB প্রশ্ন, ইত্যাদি) আক্রমনাত্মকভাবে সমান্তরাল।

### 🔄 অধিবেশনের ধারাবাহিকতা

- **Switch providers mid-session** with `UAGENT_PROVIDER` — conversation history is preserved.
- **Reload past sessions** with `:load <index>` — pick up where you left off.
- **Tool result caching** avoids redundant re-execution when the same tool call repeats.

### 🤖 অটো-পাইলট (`:অটো`)

uag **স্বয়ংক্রিয়ভাবে একাধিক LLM রাউন্ড জুড়ে একটি লক্ষ্য অনুসরণ করতে পারে**। জটিল, বহু-পদক্ষেপের কাজগুলির জন্য উপযুক্ত যেগুলির পুনরাবৃত্তিমূলক পরিমার্জন প্রয়োজন।

- **এটি কীভাবে কাজ করে**: প্রতিটি রাউন্ডে একটি প্রধান ক্যোয়ারী থাকে (ধাপ A) এর পরে একটি পর্যালোচকের রায় (ধাপ B) যা সিদ্ধান্ত নেয় "সম্পূর্ণ বা চালিয়ে যাবেন?"
- **একই প্রদানকারী, একই API**: পর্যালোচকের রায় মূল ক্যোয়ারী হিসাবে অভিন্ন কোড পাথ ব্যবহার করে — প্রতিক্রিয়া API সমর্থন সহ।
- **পৃথক বিচারক LLM** (ঐচ্ছিক): পর্যালোচকের জন্য একটি ভিন্ন প্রদানকারী/মডেল ব্যবহার করতে `UAGENT_AP_PROVIDER` সেট করুন (যেমন, বিচারের জন্য একটি সস্তা মডেল ব্যবহার করুন)।
- **যেকোনো সময় প্রস্থান করুন**: অবিলম্বে বন্ধ করতে `x` কী টিপুন, এমনকি মধ্য-প্রতিক্রিয়াও। অথবা পর্যালোচককে সিদ্ধান্ত নিতে দিন কখন লক্ষ্য পূরণ হবে।
- **কনফিগারযোগ্য**: বাজেট নিয়ন্ত্রণ করতে `--সর্বোচ্চ রাউন্ডস N`।

সম্পূর্ণ ডকুমেন্টেশনের জন্য [README_AUTO.md](https://github.com/awaku7/agentcli/blob/main/README_AUTO.md) দেখুন।

### 🧩 ব্যাচ স্টেট ম্যানেজার

uag দীর্ঘ-চলমান মাল্টি-ফাইল টাস্ক জুড়ে অগ্রগতি ট্র্যাক করতে পারে। যখন LLM কয়েক ডজন ফাইল প্রসেস করে, তখন `batch_state` ডিস্কে মুলতুবি, সম্পূর্ণ এবং ব্যর্থ ফাইলগুলির তালিকা বজায় রাখে। যদি সেশনটি শেষ হয় বা একটি রাউন্ড টাইম শেষ হয়, পরবর্তী রানটি যেখানে থামানো হয়েছিল সেখান থেকে আবার শুরু হয় - কিছুই হারিয়ে যায় না।

### 🛡 হিউম্যান-ইন-দ্য-লুপ

`human_ask` LLM কে বিরতি দিতে দেয় এবং ধ্বংসাত্মক ক্রিয়াকলাপ (ফাইল মুছে ফেলা, ওভাররাইট, শেল কমান্ড) করার আগে আপনার নিশ্চিতকরণের জন্য জিজ্ঞাসা করতে দেয়। আপনি নিয়ন্ত্রণে থাকুন।

### 🛑 ইন্টারাপ্ট (সি-কী / স্টপ বোতাম)

যেকোন সময় LLM রেসপন্স জেনারেশন বন্ধ করুন এবং LLM এ স্টপ কমান্ড ইনজেক্ট করুন।

| ইন্টারফেস | কিভাবে বাধা দিতে হয় |
|---|---|
| **CLI** | LLM স্ট্রিমিংয়ের সময় `c` কী টিপুন — বর্তমান প্রতিক্রিয়া বন্ধ হয়ে যায়, এবং `"স্টপ"` একটি ব্যবহারকারীর বার্তা হিসাবে পাঠানো হয় যাতে এলএলএম সেই অনুযায়ী প্রতিক্রিয়া জানায় |
| **ওয়েব UI** | লাল **■ স্টপ** বোতামে ক্লিক করুন (এলএলএম প্রক্রিয়াকরণের সময় স্বয়ংক্রিয়ভাবে প্রদর্শিত হবে) |
| **ডেস্কটপ GUI** | লাল **■** বোতামে ক্লিক করুন (LLM প্রক্রিয়াকরণের সময় স্বয়ংক্রিয়ভাবে প্রদর্শিত হবে) |

বিঘ্নটি "প্রম্পট ইনজেকশন" হিসাবে কাজ করে: শুধু বাদ দেওয়ার পরিবর্তে, এটি ব্যবহারকারীর বার্তা হিসাবে LLM-এ `"Stop"` ফিড করে, যা এটিকে সৌহার্দ্যপূর্ণভাবে সমাপ্তি বা বাধা স্বীকার করার অনুমতি দেয়।

অটো-পাইলট মোড থেকে প্রস্থান করতে `x` কী টিপুন (দেখুন [README_AUTO.md](https://github.com/awaku7/agentcli/blob/main/README_AUTO.md))।

### 🕵️ ব্রাউজার অটোমেশন এবং ওয়েব ইন্সপেক্টর

দুটি পরিপূরক নাট্যকার-ভিত্তিক সরঞ্জাম:

- **ব্রাউজার_প্লেওয়াইট**: প্রকৃত ব্রাউজার সেশনগুলি স্বয়ংক্রিয় করুন — নেভিগেট করুন, ক্লিক করুন, ফর্মগুলি পূরণ করুন, ডেটা বের করুন, বহু-পৃষ্ঠা প্রবাহ পরিচালনা করুন৷ মাথাবিহীন বা মাথাহীন কাজ করে।
- **নাট্যকার_ইন্সপেক্টর**: ব্রাউজার ট্রানজিশন রেকর্ড করুন, প্রতিটি ধাপে DOM স্ন্যাপশট এবং স্ক্রিনশট ক্যাপচার করুন। ওয়েব ইন্টারঅ্যাকশন ডিবাগ করার জন্য বা সময়ের সাথে পৃষ্ঠার পরিবর্তনের অডিট করার জন্য দরকারী।

### 🔄 ডায়নামিক টুল লোড হচ্ছে

`tool_catalog` এবং `tool_load` আপনাকে রানটাইমে টুলস আবিষ্কার ও সক্ষম করতে দেয়।
স্টার্টআপে সবকিছু লোড করার দরকার নেই — আপনার যখন প্রয়োজন তখনই সক্রিয় করুন।

### 🌐 i18n / L10n

### 🔒 এনক্রিপ্ট করা এনভায়রনমেন্ট ভেরিয়েবল

API কী এবং গোপনীয়তাগুলি `.env.sec`-এ সংরক্ষণ করুন — একটি এনক্রিপ্ট করা `.env` ফাইল।
`uag_envsec` দিয়ে পরিচালনা করুন।
